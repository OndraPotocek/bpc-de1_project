    XDC (Xilinx Design Constraints) file was edited specifically for the Nexys A7-50T.
    \begin{itemize}
      \item \textbf{Inputs:} System clock (clk), Buttons (debounced) for Reset and Waveform Selection. Switches mapped for Period and Duty Cycles to be set.
      \item \textbf{Outputs:} Visuals, such as standard LEDs and 7-segment displays. PWM signals' primary output routed to pin A11 (LPF) for audio output, as well as Pmod JA for oscilloscope monitoring. No need to map the GND pin.
    \end{itemize}
    There was no need to use UART and RGB LEDs, those were disabled.
